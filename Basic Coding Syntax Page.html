Basic Coding Syntax

The Syntax you need to start making a website

There are thousands of possible syntaxs that you can use. Every single one does something else. I will show you some of the basic ones so that you can get started and know som of the basic syntax.

The image on the right shows you a couple things, first it shows you that before starting to type any code you have to set the doc type to HTML. This way the computer knows that your writing code for a HTML website. There is also the h1 tag. This is the header tag, it is used often as a title for a section on your website. The header tag does not have to be h1, it can be h2,h3,h4, etc. The number is the size of the header the bigger the number the smaller the header. The second tag you can see is the p tag, this is the paragraph tag. This is used for any basic writing on your website.

The second image shows you the syntax you need to know to be able to edit specfic tags like the header or paragraph tag, it also shows you how to add an image. The img src tag is the tag to add an image, between the quotation marks you put the image you want to add to the website. The alt tag is there incase your image doesn't load the text that you put in the alt tag will appear. The style tag can be used in many ways, you use this tag to change the color, change the size, or change where the thing is positioned.
