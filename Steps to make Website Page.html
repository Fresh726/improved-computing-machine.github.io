Steps to make a Website

All the steps to make a HTML website

Step 1

The first thing you want to do before starting to make a website is to decide which type of website you want to make it about. Do you want your website to be for buisness, personal, or something else. Once you have that in mind you can start figuring out what you will do for the other steps.

Step 2

Now that you have figured out what type of website you want to make you can start thinking about what software you want to use to code the website. There are many options such as Notepad++, Atom, Sublime Text, etc. After you pick on eyou also want to get familiar with the software, for example you could look up tutorials that explain how to use the software you picked.

Step 3

After selecting a software to use you will want to actually start coding the website. Some basic syntax you need to know is on the page 'Basic Coding Syntax' on your right. Once you have finished coding the website you can start to move on to the next step

Step 4

After you finshed coding the website you will want to actually put your website on the internet. At the moment when your viewing your website only you can open that. So it is important to make it publically viewable. There are many ways to do this, the most common way is to go on a hosting website. This website will host your website on the internet so that everyone can see it.

Step 5

After putting your website on the internet you are basically done. If you want you can also put your website on Google Analytics. This will show you how many people have viewed your website and how long they have been on it. After that there is nothing else to do, you can sit back and relax while looking at your beautiful website
